Dropdown from fiscalo. Use via `window.Fiscalo.Dropdown` (bundle loaded from the root `_ds_bundle.js`).

## Props

```ts
interface DropdownProps {
  trigger: React.ReactNode;
  children?: React.ReactNode;
  align?: "left" | "right";
  width?: number;
  /** Clase del contenedor (p.ej. `desktop-only` para ocultarlo en móvil). */
  className?: string;
}
```
